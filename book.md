## บทที่ 1: บทนำสู่เทคโนโลยีเว็บและโครงสร้าง HTML เบื้องต้น

### 1.1 ความรู้ทั่วไปเกี่ยวกับระบบเว็บ (Web Technologies Introduction)
*   **Website (เว็บไซต์)**: แหล่งรวบรวมหน้าเว็บเพจที่ถูกเก็บไว้บนคอมพิวเตอร์แม่ข่าย (Web Server) ซึ่งเชื่อมโยงกันผ่านอินเทอร์เน็ต
*   **Web Page (หน้าเว็บ)**: เอกสารที่เขียนขึ้นด้วยภาษาคอมพิวเตอร์ เช่น HTML, CSS และ JavaScript
*   **Homepage (โฮมเพจ)**: หน้าแรกของเว็บไซต์ที่ทำหน้าที่เป็นดัชนีหลักในการนำทางไปยังหน้าอื่นๆ
*   **Domain Name (ชื่อโดเมน)**: ชื่อระบบอักษรที่ใช้แทน IP Address เพื่อให้ผู้ใช้จดจำได้ง่าย เช่น `google.com`
*   **Web Hosting (เว็บโฮสติ้ง)**: พื้นที่เก็บไฟล์ข้อมูลของเว็บไซต์บนเซิร์ฟเวอร์ที่เปิดทำงานตลอด 24 ชั่วโมง
*   **URL (Uniform Resource Locator)**: ที่อยู่แบบสมบูรณ์ของหน้าเว็บเพจ เช่น `https://www.example.com/about-us`

### 1.2 โครงสร้างพื้นฐานของเอกสาร HTML (HTML Document Anatomy)
ภาษา HTML (HyperText Markup Language) เป็นภาษาหลักในการวางโครงสร้างของหน้าเว็บ โดยมีโครงสร้างมาตรฐานดังนี้:

```html
<!DOCTYPE html>
<html>
<head>
    <title>ชื่อหน้าเว็บของฉัน</title>
</head>
<body>
    <h1>ยินดีต้อนรับสู่เว็บเพจแรก</h1>
    <p>นี่คือข้อความย่อหน้าแรกที่เขียนด้วยแท็ก p</p>
</body>
</html>
```

### 1.3 แท็กหลักที่ควรรู้จักในบทที่ 1
*   `<!DOCTYPE html>`: ประกาศว่าเอกสารนี้เป็นมาตรฐาน HTML5 เพื่อป้องกันเบราว์เซอร์แสดงผลเพี้ยน
*   `<html>`: แท็กเริ่มต้นและสิ้นสุดของโครงสร้าง HTML ทั้งหมด
*   `<head>`: ส่วนจัดเก็บข้อมูลเบื้องหลังของเว็บ เช่น Title, Meta set, CSS links
*   `<body>`: ส่วนเนื้อหาที่จะไปแสดงผลบนหน้าต่างเว็บเบราว์เซอร์ให้ผู้เข้าชมเห็น
*   `<h1>` ถึง `<h6>`: แท็กสำหรับระบุหัวข้อ (Heading) เรียงจากใหญ่สุดไปเล็กสุด
*   `<p>`: แท็กระบุข้อความย่อหน้า (Paragraph)
*   `<br/>`: แท็กสำหรับสั่งเว้นวรรคขึ้นบรรทัดใหม่ (Line Break) เป็นแท็กเดี่ยวไม่มีคู่ปิด
*   `<hr/>`: แท็กขีดเส้นแบ่งเนื้อหาแนวนอน (Horizontal Rule)

## บทที่ 2: มาตรฐาน HTML5, รูปแบบอักษร และลิสต์รายการ (Lists)

### 2.1 มาตรฐาน Semantic HTML5
มาตรฐานสมัยใหม่กำหนดให้ผู้สร้างเว็บเลือกใช้แท็กที่มีความหมายในตัว (Semantic) เช่น ใช้ `<h1>` เฉพาะส่วนหัวเรื่องหลักเพียงครั้งเดียวต่อหน้า และไม่ควรใช้แท็กตัวหนาหรือตัวใหญ่สะเปะสะปะเพื่อตกแต่งหน้าตา (Abuse for styling) แต่ควรใช้ CSS ในการจัดการแทน

### 2.2 แท็กจัดรูปแบบข้อความระดับเฉพาะจุด (Text Formatting)
*   `<b>` และ `<strong>`: ทำตัวหนา (strong จะสื่อความสำคัญสำหรับโปรแกรมอ่านหน้าจอด้วย)
*   `<i>` และ `<em>`: ทำตัวเอียง (em สื่อการเน้นเสียงกระแทกความสำคัญ)
*   `<mark>`: ไฮไลต์พื้นหลังข้อความสีเหลือง
*   `<small>`: แสดงอักษรขนาดเล็กกว่าปกติ
*   `<del>`: ขีดฆ่าข้อความแสดงข้อความที่ถูกยกเลิก (Deleted text)
*   `<ins>`: ขีดเส้นใต้แสดงข้อความที่แทรกใหม่ (Inserted text)
*   `<sub>`: ทำตัวห้อยด้านล่าง (Subscript) เช่น ในสูตรทางเคมี: `H<sub>2</sub>O` -> H₂O
*   `<sup>`: ทำตัวยกด้านบน (Superscript) เช่น ในเลขยกกำลัง: `x<sup>2</sup>` -> x²

### 2.3 การสร้างรายการลำดับข้อความ (HTML Lists)
ภาษา HTML แบ่งกลุ่มรายการออกเป็น 3 แบบ:
1.  **Unordered List (รายการไม่มีลำดับ)**: ใช้จุดกลมนำหน้าหัวข้อ
    ```html
    <ul>
        <li>กาแฟ</li>
        <li>ชา</li>
    </ul>
    ```
2.  **Ordered List (รายการมีลำดับ)**: ใช้ตัวเลขหรือตัวอักษรเรียงกัน
    ```html
    <ol>
        <li>ตื่นนอน</li>
        <li>แปรงฟัน</li>
    </ol>
    ```
3.  **Description List (รายการคำอธิบายศัพท์)**: เหมาะสำหรับพจนานุกรม
    ```html
    <dl>
        <dt>HTML</dt>
        <dd>ภาษาหลักที่ใช้สร้างโครงสร้างหน้าเว็บ</dd>
    </dl>
    ```

## บทที่ 3: สื่อรูปภาพ ลิงก์เชื่อมโยง ตารางข้อมูล และการต่อยอดด้วย CSS

### 3.1 การใส่ภาพประกอบหน้าเว็บ (HTML Images)
เราจะใช้แท็กเดี่ยว `<img>` ในการแสดงผลรูปภาพ โดยมี Attributes ที่สำคัญคือ:
*   `src`: ที่อยู่ไฟล์ภาพ (Relative path หรือ Absolute URL)
*   `alt`: คำอธิบายรูปภาพสำหรับผู้พิการทางสายตา และช่วยเรื่อง SEO ของเว็บ
*   `width` / `height`: กำหนดขนาดกว้างและสูงเป็นพิกเซล
*   `loading="lazy"`: สั่งให้โหลดภาพเฉพาะเมื่อผู้ใช้เลื่อนจอไปถึง ช่วยให้เว็บโหลดหน้าแรกเร็วขึ้น

```html
<img src="images/banner.jpg" alt="แบนเนอร์แสดงโปรโมชั่นสินค้า" width="800" height="400" loading="lazy">
```

### 3.2 การสร้างลิงก์เชื่อมโยง (Anchor Tag)
แท็ก `<a>` ใช้สำหรับพาผู้ใช้ไปยังปลายทางต่างๆ:
*   **ลิงก์ภายนอก**: `<a href="https://google.com" target="_blank" rel="noopener">ค้นหาข้อมูล</a>` (ใช้ `target="_blank"` เพื่อเปิดหน้าต่างใหม่)
*   **ลิงก์ส่งอีเมล**: `<a href="mailto:test@example.com">ติดต่อผู้ดูแลระบบ</a>`
*   **ลิงก์โทรศัพท์**: `<a href="tel:+66812345678">โทรหาเรา</a>`

### 3.3 โครงสร้างตาราง HTML (Tables)
การจัดเรียงข้อมูลแบบตารางประกอบด้วยส่วนประกอบ:
*   `<table>`: ครอบขอบเขตตาราง
*   `<caption>`: ชื่อหรือคำบรรยายหัวข้อตาราง
*   `<tr>` (Table Row): แถวในตาราง
*   `<th>` (Table Header): เซลล์ส่วนหัวข้อแถว/คอลัมน์ (แสดงผลตัวหนาและอยู่กึ่งกลาง)
*   `<td>` (Table Data): เซลล์เก็บข้อมูลปกติ

### 3.4 วิธีการต่อใช้งาน CSS 3 รูปแบบ
1.  **Inline CSS**: เขียนลงในแอทริบิวต์ style ของแท็กโดยตรง
    `<p style="color: blue;">ข้อความสีน้ำเงิน</p>`
2.  **Internal CSS**: เขียนรวบรวมไว้ที่หัวหน้าเพจในแท็ก `<style>`
    `<style> body { background-color: f4f4f4; } </style>`
3.  **External CSS**: แยกเขียนไฟล์ `.css` ต่างหาก แล้วดึงลิงก์เข้ามาใช้ใน `<head>`
    `<link rel="stylesheet" href="style.css">`

## บทที่ 4: การควบตารางข้อมูลขั้นสูง และแบบฟอร์มรับข้อมูล (Forms)

### 4.1 การควบเซลล์ตารางด้วย colspan และ rowspan
*   `colspan`: ขยายเซลล์ปัจจุบันไปครอบคลุมคอลัมน์ด้านข้าง (แนวนอน) เช่น `colspan="2"`
*   `rowspan`: ขยายเซลล์ปัจจุบันไปครอบคลุมแถวด้านล่าง (แนวตั้ง) เช่น `rowspan="2"`

### 4.2 แบบฟอร์ม HTML (Forms)
แท็ก `<form>` เป็นส่วนประสานที่รับข้อมูลจากฝั่งผู้ใช้งานเพื่อจัดส่งไปประมวลผลต่อยัง Web Server
*   `action`: ระบุ URL ปลายทางที่ต้องการส่งข้อมูลไปประมวลผล
*   `method`:
    *   **GET**: ส่งข้อมูลแนบไปกับแถบ URL เหมาะกับการค้นหา ไม่ปลอดภัยสำหรับข้อมูลส่วนตัว
    *   **POST**: ส่งข้อมูลซ่อนไปใน Body ของ Request เหมาะสำหรับรหัสผ่านและการลงทะเบียน

### 4.3 แท็กและคอนโทรลสำคัญในฟอร์ม
*   `<input type="text">`: กล่องรับข้อมูลบรรยายบรรทัดเดียว
*   `<input type="password">`: กล่องรับรหัสผ่าน (ซ่อนอักษรเป็นจุดกลม)
*   `<input type="radio">`: ปุ่มวิทยุสำหรับเลือกได้เพียงตัวเลือกเดียวจากกลุ่ม (ต้องมี `name` เหมือนกันทุกปุ่ม)
*   `<input type="checkbox">`: กล่องสำหรับติ๊กเลือกได้หลายข้อความพร้อมกัน
*   `<select>` และ `<option>`: รายการเลือกแบบ Dropdown
*   `<textarea>`: กล่องป้อนข้อความขนาดใหญ่หลายบรรทัด
*   `<fieldset>` และ `<legend>`: กรอบจัดหมวดหมู่ชุดกรอกข้อมูลเพื่อความสวยงามและเข้าถึงง่าย

## บทที่ 5: การประยุกต์ใช้งาน IDE VS Code, วิดีโอ/เสียง และ CSS Selectors

### 5.1 การจัดสภาพแวดล้อมระดับนักพัฒนาใน VS Code
*   **Live Server**: รันเว็บบน localhost ส่วนตัว และอัปเดตหน้าแสดงผลบนเบราว์เซอร์อัตโนมัติเมื่อกดเซฟโค้ด
*   **คีย์ลัดน่ารู้**:
    *   `Shift + Alt + F`: ปรับโค้ดให้ยุบหยัก สวยงาม เรียงตามลำดับความลึกอัตโนมัติ (Format Document)
    *   `Ctrl + /`: เปิด/ปิดข้อความคอมเมนต์ (Comment) ในบรรทัดที่ปัก cursor

### 5.2 การแทรกไฟล์วิดีโอและเสียง (Media Elements)
*   **แท็กวิดีโอ `<video>`**:
    ```html
    <video width="640" height="360" controls poster="thumbnail.jpg">
        <source src="movie.mp4" type="video/mp4">
        เบราว์เซอร์ของคุณไม่รองรับไฟล์วิดีโอนี้
    </video>
    ```
*   **แท็กเสียง `<audio>`**:
    ```html
    <audio controls loop>
        <source src="song.mp3" type="audio/mpeg">
    </audio>
    ```

### 5.3 องค์ประกอบ Div/Span และ CSS Selectors
*   `<div>` (Block Element): สัดส่วนกล่องที่จะขึ้นบรรทัดใหม่ทันที และยึดพื้นที่ความกว้างเต็มหน้าจอเสมอ
*   `<span>` (Inline Element): รอยต่อจัดกลุ่มข้อความภายในบรรทัด ยึดพื้นที่เท่าขนาดเนื้อหาจริง ไม่เว้นบรรทัดใหม่
*   **CSS Selectors (ตัวระบุสไตล์)**:
    1.  *Element Selector*: ระบุทุกแท็กชนิดนั้น เช่น `p { color: gray; }`
    2.  *Class Selector*: ระบุคลาสสไตล์ที่สร้างขึ้น ซ้ำกันได้ในหน้าเพจ โดยมีจุด (.) นำหน้า เช่น `.btn-red { color: red; }`
    3.  *ID Selector*: ระบุเป้าหมายเจาะจง ห้ามซ้ำกันเด็ดขาดในหน้าเพจ โดยมีชาร์ป (#) นำหน้า เช่น `#main-header { font-size: 24px; }`

## บทที่ 6: CSS Typography และกฎของแบบจำลองกล่อง (Box Model)

### 6.1 การควบคุมรูปแบบตัวอักษร (Typography Properties)
*   `font-family`: เลือกแบบฟอนต์ของตัวอักษร นิยมกำหนดแบบตระกูลลำรอง เช่น `'Prompt', 'Sarabun', sans-serif`
*   `font-size`: กำหนดขนาด เช่น `16px` หรือแบบยืดหยุ่น `1.2rem`
*   `text-align`: จัดแนวข้อความ (`left`, `center`, `right`, `justify`)
*   `text-decoration`: เพิ่ม/ลบเส้นใต้ตัวอักษร เช่น `none` (นิยมใช้ลบเส้นใต้ลิงก์ `<a>`) หรือ `line-through` (ขีดฆ่าราคาสินค้า)
*   `text-shadow`: ใส่เงาให้อักษร เช่น `2px 2px 4px rgba(0,0,0,0.3)`
*   **การย่อข้อความด้วยจุดไข่ปลา**:
    ```css
    .ellipsis-text {
        white-space: nowrap;
        overflow: hidden;
        text-overflow: ellipsis;
    }
    ```

### 6.2 แบบจำลองกล่อง CSS Box Model
ทุกออบเจกต์ใน HTML จะถูกมองเป็นกล่องสี่เหลี่ยมที่มีโครงสร้างความหนา 4 ชั้น:
1.  **Content (เนื้อหาหลัก)**: พื้นที่ขอบเขตสิ่งของ ขนาดกว้าง/สูงควบคุมด้วย `width` และ `height`
2.  **Padding (ระยะห่างภายใน)**: พื้นที่เบาะฟองน้ำกันกระแทกระหว่างตัวเนื้อหากับขอบกล่อง
3.  **Border (เส้นขอบกล่อง)**: ความหนา ลวดลาย และสีของกรอบกล่อง
4.  **Margin (ระยะห่างภายนอก)**: พื้นที่ว่างเว้นห่างระหว่างกล่องนี้กับกล่องอื่นๆ ข้างเคียง

```
 ┌───────────────────────────────────────────┐
 │                Margin                     │
 │   ┌───────────────────────────────────┐   │
 │   │            Border                 │   │
 │   │   ┌───────────────────────────┐   │   │
 │   │   │        Padding            │   │   │
 │   │   │   ┌───────────────────┐   │   │   │
 │   │   │   │    Content        │   │   │   │
 │   │   │   │  (Width x Height) │   │   │   │

 │   │   └───────────────────────────┘   │   │
 │   └───────────────────────────────────┘   │
 └───────────────────────────────────────────┘
```

## บทที่ 7: ระบบจัดหน้าเว็บเพจยืดหยุ่นด้วย CSS Flexbox

### 7.1 Flexbox คอนเซปต์หลัก
ระบบ **Flexbox** (Flexible Box Layout) คือระบบการจัดวาง Layout แบบ 1 มิติ (จัดการในแนวแกนแถวหรือแกนคอลัมน์อย่างใดอย่างหนึ่ง) ออกแบบมาเพื่อให้สิ่งของภายในกล่องยืดหยุ่นและเปลี่ยนตำแหน่งตามขนาดของหน้าจอได้ง่าย

### 7.2 คำสั่งหลักของ Flex Container (กล่องแม่)
*   `display: flex`: สั่งให้กล่องนี้เปลี่ยนพฤติกรรมเป็นระบบ Flexbox
*   `flex-direction`: ทิศทางแกนหลัก
    *   `row` (ดีฟอลต์): จัดเรียงตามแนวนอนจากซ้ายไปขวา
    *   `column`: จัดเรียงตามแนวตั้งจากบนลงล่าง
*   `justify-content`: จัดการกระจายตัวของสิ่งของในแนวแกนหลัก (Main Axis)
    *   `flex-start`: ชิดจุดเริ่มต้น
    *   `center`: อยู่กึ่งกลาง
    *   `space-between`: เว้นระยะช่องว่างระหว่างกล่องเท่าๆ กัน โดยกล่องแรกและสุดท้ายติดขอบ
    *   `gap`: กำหนดช่องว่างระยะห่างคงที่ระหว่างกล่อง เช่น `gap: 20px`
*   `align-items`: จัดการตำแหน่งสิ่งของในแกนรอง (Cross Axis)
    *   `flex-start` / `center` / `flex-end` / `stretch` (ยืดเต็มขนาดกล่องแม่)
*   `flex-wrap`: สั่งให้ย้ายสิ่งของขึ้นแถวใหม่เมื่อพื้นที่กล่องแม่ไม่เพียงพอ
    *   `nowrap` (ดีฟอลต์): บีบกล่องย่อยให้ลีบแต่อยู่แถวเดิม
    *   `wrap`: สิ่งของจะไหลขึ้นบรรทัดใหม่เมื่อเบียดกันจนหมดพื้นที่
