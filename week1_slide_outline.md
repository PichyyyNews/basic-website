## Slide 2: วัตถุประสงค์การเรียนรู้
*   อธิบายส่วนประกอบต่าง ๆ ของเว็บไซต์ได้ (Web Page, Domain, Hosting, URL)
*   บอกความแตกต่างระหว่าง Static และ Dynamic website ได้
*   เขียนโครงสร้างเอกสารภาษา HTML5 เบื้องต้นได้
*   สร้างหน้าเว็บแรกและเปิดแสดงผลผ่านเบราว์เซอร์ได้สำเร็จ

## Slide 3: Website คืออะไร?
*   **Website (เว็บไซต์):** กลุ่มของหน้าเว็บเพจที่เชื่อมโยงถึงกัน จัดเก็บไว้บนคอมพิวเตอร์เซิร์ฟเวอร์ (Web Server)
*   **Web Page (หน้าเว็บเพจ):** เอกสารแต่ละหน้าที่เราเปิดอ่าน เช่น หน้าข่าว, หน้าสินค้า เขียนขึ้นด้วยภาษาคอมพิวเตอร์
*   **Homepage (โฮมเพจ):** หน้าแรกสุดของเว็บไซต์ เปรียบเสมือนหน้าร้านหรือสารบัญหลัก

## Slide 4: ส่วนประกอบที่ทำให้เว็บออนไลน์ได้
*   **Domain Name (ชื่อโดเมน):** ชื่อแทน IP Address เพื่อให้เรียกง่าย เช่น `google.com`, `wikipedia.org`
*   **Web Hosting (เว็บโฮสติ้ง):** เซิร์ฟเวอร์ที่เปิดทิ้งไว้ 24 ชม. เพื่อฝากไฟล์เว็บของเราให้คนทั่วโลกมาดูได้
*   **URL (Uniform Resource Locator):** ลิงก์ที่อยู่แบบสมบูรณ์ของหน้าเว็บ เช่น `https://www.example.com/about-us`

## Slide 5: Static vs Dynamic Website
*   **Static Website:**
    *   แสดงผลข้อมูลคงที่ เขียนโค้ด HTML/CSS ไว้แบบไหนแสดงผลแบบนั้น
    *   แก้ไขยาก ต้องแก้ในโค้ดตรงๆ ไม่มีระบบหลังบ้านหรือฐานข้อมูล
*   **Dynamic Website:**
    *   แสดงข้อมูลเปลี่ยนแปลงตามผู้ใช้ เช่น Facebook, Shopee
    *   ดึงข้อมูลจากฐานข้อมูล มีระบบหลังบ้านช่วยจัดการประมวลผล

## Slide 6: ภาษาที่ใช้สร้างเว็บไซต์
*   **HTML (HyperText Markup Language):** โครงกระดูก / โครงสร้างหน้าเว็บ
*   **CSS (Cascading Style Sheets):** เสื้อผ้า / การตกแต่ง สีสัน สไตล์
*   **JavaScript:** กล้ามเนื้อ / ระบบคำนวณและพฤติกรรมโต้ตอบ

## Slide 7: เครื่องมือแรกของเรา
*   **Windows Notepad:** โปรแกรมพิมพ์ข้อความที่แถมมากับ Windows สำหรับหัดเขียนโค้ดในสัปดาห์แรกๆ
*   **Web Browser (Chrome/Edge):** โปรแกรมสำหรับประมวลผลโค้ด HTML มาแสดงเป็นหน้าเว็บสวยงาม

## Slide 8: โครงสร้างหลักของ HTML5
*   แท็กหลักที่ขาดไม่ได้:
    ```html
    <!DOCTYPE html>
    <html>
    <head>
        <title>ชื่อเพจ</title>
    </head>
    <body>
        เนื้อหาที่จะแสดงผล
    </body>
    </html>
    ```

## Slide 9: แท็กข้อความเบื้องต้น
*   `<h1>` ถึง `<h6>`: หัวเรื่องขนาดต่างๆ (Heading)
*   `<p>`: ย่อหน้าข้อความ (Paragraph)
*   `<br/>`: สั่งขึ้นบรรทัดใหม่ (Line Break)
*   `<hr/>`: เส้นแบ่งหัวข้อแนวนอน (Horizontal Rule)
