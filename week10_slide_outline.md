# โครงร่างสไลด์สัปดาห์ที่ 10: การจัดโครงสร้างเลย์เอาต์ขั้นสูงด้วย CSS Grid Layout

สไลด์ชุดนี้ใช้สำหรับบรรยายและสาธิตการเขียนโค้ดรายสัปดาห์ที่ 10 (บรรยาย 1 ชั่วโมง, ปฏิบัติ 3 ชั่วโมง) โดยเรียงแบบ “คำสั่งหนึ่งคำสั่งต่อหนึ่งแนวคิด” เพื่อให้นักเรียนพิมพ์ตามและเห็นผลลัพธ์ทันที

---

## Slide 1: หัวข้อหลัก
* **หัวข้อ:** การจัดโครงสร้างเลย์เอาต์ขั้นสูงด้วย CSS Grid Layout
* **วิชา:** 21909-1006 พื้นฐานการสร้างเว็บไซต์ (ปวช. 1)

## Slide 2: แผนการเรียนรู้วันนี้
* เริ่มจาก HTML เดิม แล้วเปิด Grid ด้วย `display: grid`
* สร้างคอลัมน์ แถว และช่องว่างทีละคำสั่ง
* ขยายพื้นที่ด้วย Grid Line และ `span`
* ทำ Gallery Grid ที่ปรับตามหน้าจอ และสรุปเป็นชิ้นงาน

## Slide 3: ก่อนเขียน CSS Grid ต้องมีอะไรบ้าง?
* แนะนำ Grid Container และ Grid Items จากโครงสร้าง HTML ตัวอย่าง
* อธิบายคำศัพท์ Column, Row, Cell และ Grid Line

## Slide 4: ขั้นที่ 1 - `display: grid`
* โค้ดสั้นสำหรับเปลี่ยนกล่องแม่เป็น Grid Container
* ผลลัพธ์เมื่อยังไม่กำหนดคอลัมน์ และข้อควรระวังเรื่องกล่องแม่/กล่องลูก

## Slide 5: ขั้นที่ 2 - `grid-template-columns`
* ตัวอย่าง `1fr 1fr 1fr`
* อธิบายหน่วย `fr` และภาพผลลัพธ์ 3 คอลัมน์

## Slide 6: แบ่งสัดส่วนคอลัมน์ด้วย `fr`
* เปรียบเทียบ `1fr 2fr 1fr` กับขนาดตายตัว `220px 1fr 160px`
* ชี้ให้เห็นว่า `fr` แบ่งเฉพาะพื้นที่ว่างที่เหลือ

## Slide 7: ขั้นที่ 3 - `repeat()`
* เปลี่ยน `1fr 1fr 1fr` เป็น `repeat(3, 1fr)`
* ยกตัวอย่างรูปแบบ `repeat(4, 1fr)` และ `repeat(2, 200px)`

## Slide 8: ขั้นที่ 4 - `grid-template-rows` และ `grid-auto-rows`
* สร้างแถวสูงเท่ากันด้วย `grid-template-rows`
* อธิบายแถวอัตโนมัติและตัวอย่าง `grid-auto-rows: 160px`

## Slide 9: ขั้นที่ 5 - `gap`
* เพิ่มระยะห่างระหว่าง Grid Items ด้วย `gap: 16px`
* เปรียบเทียบ `gap`, `row-gap` และ `column-gap`

## Slide 10: ขั้นที่ 6 - Grid Line และ `grid-column`
* อธิบายเส้นกริดของ Grid 3 คอลัมน์
* สาธิต `grid-column: 1 / 3` และผลลัพธ์ที่ Item กิน 2 คอลัมน์

## Slide 11: ขั้นที่ 7 - `span`
* สาธิต `grid-column: span 2` และ `grid-row: span 2`
* แสดงภาพเด่นแบบ 2 คอลัมน์ × 2 แถว

## Slide 12: ตั้งชื่อพื้นที่ด้วย `grid-template-areas`
* ตัวอย่าง Header, Sidebar, Main และ Footer
* เชื่อมชื่อพื้นที่กับ `grid-area`

## Slide 13: ขั้นที่ 8 - `minmax()`
* กำหนดความกว้างขั้นต่ำและการขยายของคอลัมน์ด้วย `minmax(180px, 1fr)`
* อธิบายว่าทำไมจึงอ่านง่ายกว่าคอลัมน์แคบเกินไป

## Slide 14: ขั้นที่ 9 - `auto-fit`
* สาธิต `repeat(auto-fit, minmax(180px, 1fr))`
* เปรียบเทียบผลลัพธ์บนหน้าจอกว้างและหน้าจอแคบ

## Slide 15: ขั้นที่ 10 - Media Query สำหรับมือถือ
* คืนภาพที่กินหลายช่องให้กลับมาเป็น 1 ช่อง
* สาธิต `@media (max-width: 480px)`

## Slide 16: รวมคำสั่งเป็น CSS ของ Gallery Grid
* แสดง CSS ฉบับใช้งานจริง: Grid, responsive columns, rows, gap และ span
* อธิบาย `object-fit: cover` สำหรับภาพเต็มกรอบ

## Slide 17: HTML ของ Gallery Grid
* แสดงโครงสร้าง `<main>`, `<article>` และ `<img>`
* เน้นการตั้งคลาสและข้อความ `alt`
