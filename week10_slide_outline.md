# โครงร่างสไลด์สัปดาห์ที่ 10: การจัดโครงสร้างเลย์เอาต์ขั้นสูงด้วย CSS Grid Layout

สไลด์ชุดนี้ใช้สำหรับประกอบการบรรยายรายสัปดาห์ที่ 10 (บรรยาย 1 ชั่วโมง, ปฏิบัติ 3 ชั่วโมง)

---

## Slide 1: หัวข้อหลัก
* **หัวข้อ:** การจัดโครงสร้างเลย์เอาต์แบบ 2 มิติด้วย CSS Grid Layout
* **วิชา:** 21909-1006 พื้นฐานการสร้างเว็บไซต์ (ปวช. 1)

## Slide 2: วัตถุประสงค์การเรียนรู้
* อธิบายความแตกต่างระหว่าง Flexbox แบบ 1 มิติ และ Grid แบบ 2 มิติได้
* ระบุบทบาทของ Grid Container และ Grid Items ได้
* สร้างคอลัมน์และแถวด้วย `grid-template-columns`, `grid-template-rows` และ `gap` ได้
* ควบคุมขนาดและตำแหน่งของ Grid Items ด้วย `grid-column`, `grid-row` และ `span` ได้
* สร้างแกลเลอรีรูปภาพที่ปรับจำนวนคอลัมน์ตามหน้าจอได้

## Slide 3: เมื่อเลย์เอาต์ต้องควบคุมทั้งแถวและคอลัมน์
* Flexbox เหมาะกับการจัดเรียงในทิศทางเดียว เช่น เมนูนำทางหรือแถวปุ่ม
* CSS Grid เหมาะกับหน้าแกลเลอรี แดชบอร์ด และหน้าเว็บที่มีพื้นที่หลายขนาด
* Grid ช่วยกำหนดโครงร่างทั้งแถวและคอลัมน์ได้จากกล่องแม่เพียงจุดเดียว

## Slide 4: Grid Container และ Grid Items
* **Grid Container:** กล่องแม่ที่เปิดใช้งานด้วย `display: grid;`
* **Grid Items:** กล่องลูกโดยตรงที่ถูกวางลงในช่องตารางอัตโนมัติ
* คำสำคัญ: Grid Line, Grid Track, Grid Cell และ Grid Area

## Slide 5: สร้างคอลัมน์และแถวด้วยหน่วย `fr`
* `grid-template-columns` ใช้กำหนดจำนวนและสัดส่วนคอลัมน์
* `1fr` คือ 1 ส่วนของพื้นที่ว่างที่เหลือ และ `2fr` กว้างเป็น 2 ส่วน
* `repeat(3, 1fr)` ช่วยสร้างคอลัมน์ซ้ำ 3 ช่องได้กระชับ
* `grid-template-rows` ใช้กำหนดความสูงของแถวในเลย์เอาต์ที่มีโครงสร้างชัดเจน

## Slide 6: `gap` และการวางตำแหน่งด้วย `span`
* `gap` สร้างระยะห่างระหว่างทุกช่องโดยไม่ต้องกำหนด `margin` ทีละกล่อง
* `grid-column: span 2;` ทำให้กล่องกินพื้นที่ 2 คอลัมน์
* `grid-row: span 2;` ทำให้กล่องกินพื้นที่ 2 แถว
* ใช้ร่วมกันเพื่อสร้างภาพเด่นหรือการ์ดข้อมูลสำคัญ

## Slide 7: Grid Area ทำให้หน้าเว็บอ่านง่าย
* `grid-template-areas` ตั้งชื่อพื้นที่ เช่น `header`, `sidebar`, `main` และ `footer`
* อ่านโครงสร้างหน้าเว็บจาก CSS ได้ทันที และแก้ไขตำแหน่งได้สะดวก
* เหมาะกับเลย์เอาต์เว็บไซต์ที่มีส่วนประกอบหลักชัดเจน

## Slide 8: สร้างกริดที่ปรับจำนวนคอลัมน์เอง
* `repeat(auto-fit, minmax(180px, 1fr))` ช่วยให้กริดพอดีกับพื้นที่จอโดยอัตโนมัติ
* `minmax()` กำหนดความกว้างขั้นต่ำที่อ่านง่าย และให้กล่องขยายเติมพื้นที่ที่เหลือ
* ใช้ Media Query ยกเลิก `span` ของภาพเด่นเมื่อหน้าจอแคบมาก

## Slide 9: สรุปและการปฏิบัติการ LAB 10
* เปิดใช้กริดด้วย `display: grid;` แล้วกำหนดโครงสร้างด้วยแถว คอลัมน์ และ `gap`
* ใช้ `fr`, `repeat()`, `auto-fit` และ `minmax()` สร้างกริดที่ยืดหยุ่น
* ใช้ `grid-column`, `grid-row` และ `span` สร้างภาพหลายขนาด
* สร้างไฟล์ `gallery-grid.html` สำหรับแกลเลอรีอย่างน้อย 8 ภาพ พร้อมทดสอบ Desktop และ Mobile
