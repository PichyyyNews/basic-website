## Slide 3: ยินดีต้อนรับสู่ Visual Studio Code
*   IDE ระดับมาตรฐานอุตสาหกรรมสำหรับการพัฒนาซอฟต์แวร์และเขียนหน้าเว็บ
*   **ทำไมต้องใช้?** 
    *   มีตัวช่วยทำสีตามโครงสร้าง (Syntax Highlighting) ดูง่าย

    *   ระบบ Extension ปลั๊กอินทุ่นแรง เช่น Live Server รันเว็บออนไลน์จำลองทันที

## Slide 4: แป้นลัดคีย์บอร์ดที่จำเป็น (Keyboard Shortcuts)
*   `Ctrl + P`: ค้นหาและเปิดไฟล์ในโครงการอย่างรวดเร็ว
*   `Ctrl + B`: ซ่อน/แสดง แถบไฟล์เมนูด้านซ้ายมือ
*   `Ctrl + /`: ทำคอมเมนต์ในโค้ด (Line Comment)
*   `Shift + Alt + F`: ปรับการจัดเรียงรอยหยักโค้ดให้สะอาดเรียบร้อยอัตโนมัติ (Format Document)

## Slide 5: แท็กเล่นไฟล์วิดีโอ `<video>` ใน HTML5
*   แทรกลิงก์วิดีโอออนไลน์หรือในเครื่องโดยไม่ต้องติดตั้งโปรแกรมเสริมภายนอก
*   **คุณสมบัติแอตทริบิวต์หลัก:**
    *   `controls`: แสดงแถบควบคุม (ปุ่มเล่น, หยุด, ระดับเสียง)
    *   `poster`: ภาพหน้าปกก่อนกดเล่นวิดีโอ
    *   `loop` / `muted`: เล่นวนซ้ำ / ปิดเสียงตั้งแต่ต้น
    *   `preload="metadata"`: โหลดเฉพาะความกว้างยาวความยาววิดีโอมารอไว้ ช่วยประหยัดอินเทอร์เน็ต

## Slide 6: แท็กเล่นไฟล์เสียง `<audio>` ใน HTML5
*   แทรกไฟล์เสียงหรือเพลงประกอบหน้าเว็บ
*   *ตัวอย่างโครงสร้างมาตรฐาน:*
    ```html
    <audio controls loop>
        <source src="sound.mp3" type="audio/mpeg">
        <source src="sound.ogg" type="audio/ogg">
        เบราว์เซอร์ของคุณไม่รองรับไฟล์เสียงนี้
    </audio>
    ```

## Slide 7: แท็ก <div> (Division)
*   **พฤติกรรม:** เป็นแท็กประเภท **Block Element**
    *   จะขึ้นบรรทัดใหม่ทันทีเมื่อเรียกใช้งาน
    *   จองพื้นที่ความกว้างแนวนอน 100% เต็มจอเสมอ
*   **วัตถุประสงค์:** ใช้จัดเลย์เอาต์โครงสร้างบ้าน แบ่งชั้น ขวา ซ้าย หัวเว็บ ท้ายเว็บ

## Slide 8: แท็ก <span>
*   **พฤติกรรม:** เป็นแท็กประเภท **Inline Element**
    *   ไม่ขึ้นบรรทัดใหม่ จะต่อทอดไปตามยาวข้อความในบรรทัดเดียวกัน
    *   จองความกว้างพื้นที่เท่าขนาดความยาวของเนื้อหาจริง
*   **วัตถุประสงค์:** ใช้จัดกลุ่มคำสั้นๆ เพื่อเปลี่ยนสีสันหรือเน้นย้ำเฉพาะบางจุดกลางประโยค

## Slide 9: ทำความเข้าใจ CSS Selector
*   ตัวเลือกคำสั่งในฝั่ง CSS เพื่อป้อนข้อมูลกำหนดว่าจะสไตล์ตัวแท็กจุดไหนของ HTML
*   **ตัวเลือกพื้นฐาน:**
    1.  **Element Selector:** เลือกตามชื่อชนิดแท็กทั้งหมด เช่น `p { color: red; }`
    2.  **ID Selector (#):** เจาะจงระบุกล่องเดียวในหน้าเพจ ห้ามซ้ำกัน เช่น `#header { background: gray; }`
    3.  **Class Selector (.):** กลุ่มสไตล์ที่สร้างขึ้นมาใช้ซ้ำได้หลายแห่ง เช่น `.text-bold { font-weight: bold; }`
