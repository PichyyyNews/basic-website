# คู่มือปฏิบัติการ Week 1: บทนำสู่เว็บไซต์และโครงสร้าง HTML เบื้องต้น

ในสัปดาห์นี้เราจะเริ่มเรียนรู้วิธีการจัดทำไฟล์หน้าเว็บแรกของเราด้วยตนเองโดยใช้เครื่องมืออย่าง Notepad และเรียนรู้ชุดแท็กที่สำคัญของภาษา HTML5

---

## 1. วิธีแสดงนามสกุลไฟล์บนเครื่องคอมพิวเตอร์ (Windows)
ก่อนเริ่มสร้างไฟล์โค้ด เราจำเป็นต้องเปิดการแสดงนามสกุลไฟล์บนระบบปฏิบัติการ Windows เสียก่อน เพื่อป้องกันการบันทึกไฟล์เป็นชื่อซ้อน เช่น `index.html.txt`

### ขั้นตอนการเปิดใช้งาน:
1. เปิด **File Explorer** ขึ้นมา (กดปุ่ม `Windows + E` บนคีย์บอร์ด)
2. คลิกเลือกที่แถบเมนู **View** ที่อยู่ด้านบนของหน้าต่าง
3. เลือกหัวข้อ **Show**
4. ติ๊กเครื่องหมายหน้าคำว่า **File name extensions** (นามสกุลไฟล์) เพื่อเปิดการแสดงผล

---

## 2. ขั้นตอนการสร้างไฟล์หน้าเว็บแรก (`index.html`)

เมื่อเปิดแสดงนามสกุลไฟล์แล้ว ให้ทำตามขั้นตอนดังต่อไปนี้เพื่อสร้างไฟล์เว็บ:
1. คลิกขวาบนพื้นที่ว่างในโฟลเดอร์ของตนเอง เลือก **New** -> **Text Document**
2. ตั้งชื่อไฟล์และเปลี่ยนนามสกุลไฟล์จาก `New Text Document.txt` ให้เป็น `index.html`
3. จะมีกล่องข้อความเตือนเรื่องการเปลี่ยนนามสกุล ให้กด **Yes**
4. คลิกขวาที่ไฟล์ `index.html` เลือก **Open with** -> **Notepad** (สำหรับพิมพ์โค้ด)
5. ดับเบิลคลิกไฟล์ `index.html` ปกติเพื่อเปิดรันแสดงผลผ่าน **Google Chrome** หรือ **Microsoft Edge**

---

## 3. โครงสร้างหลักของ HTML5 และแท็กพื้นฐาน

เมื่อเปิดไฟล์ด้วย Notepad แล้ว ให้พิมพ์โค้ดโครงสร้างมาตรฐานลงไปดังนี้:

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>หน้าแรกของฉัน</title>
</head>
<body>
    <h1>สวัสดี ยินดีต้อนรับเข้าสู่เว็บไซต์แรก!</h1>
    <hr/>
    <p>นี่คือข้อความบรรยายสั้น ๆ เกี่ยวกับตัวของฉัน เขียนด้วยแท็กย่อหน้า p ซึ่งสามารถรองรับภาษาไทยได้เป็นอย่างดีเพราะระบุระบบถอดรหัสเป็น UTF-8 ไว้ในส่วนหัว</p>
    <br/>
    <p>หากเราต้องการแยกข้อความเป็นอีกย่อหน้าหนึ่ง ก็พิมพ์แท็ก p เพิ่มเติมขึ้นมาใหม่</p>
</body>
</html>
```

### คำอธิบายแท็ก (Tag Explanation):
*   `<!DOCTYPE html>`: ใช้ประกาศมาตรฐาน HTML5
*   `<html lang="th">`: แท็กหลักที่ครอบคลุมเนื้อหาเว็บทั้งหมด ระบุว่าเป็นภาษาไทย
*   `<head>`: บันทึกข้อมูลเบื้องหลังของหน้าเพจ (Meta Data)
*   `<meta charset="UTF-8">`: ป้องกันไม่ให้ภาษาไทยแสดงผลเป็นเครื่องหมายคำถามหรืออักษรต่างดาว (ภาษาไทยเพี้ยน)
*   `<title>`: ข้อความที่ปรากฏบนแท็บหัวเรื่องของเว็บเบราว์เซอร์
*   `<body>`: เนื้อหาทั้งหมดที่จะแสดงบนหน้าจอ
*   `<h1>` ถึง `<h6>`: แสดงผลข้อความหัวเรื่อง ขนาดใหญ่ที่สุดคือ `<h1>` และเล็กที่สุดคือ `<h6>`
*   `<p>`: กำหนดกรอบของย่อหน้าข้อความ
*   `<br/>`: แท็กขึ้นบรรทัดใหม่เฉพาะจุด (ไม่ต้องปิดแท็กคู่)
*   `<hr/>`: สร้างเส้นคั่นแนวนอนสำหรับแยกเนื้อหา

## 📝 ใบงานปฏิบัติท้ายบทเรียน (Assignment 1)

**โจทย์:** ให้นักเรียนสร้างไฟล์หน้าเว็บ `my-profile.html` แนะนำตัวผู้เรียนโดยใช้เนื้อหาจริง และใช้แท็กที่เรียนมาจัดหน้าต่าง ๆ ดังนี้:
1. หัวเรื่องใหญ่สุด (`<h1>`) แสดงชื่อ-นามสกุลจริงของผู้เรียน
2. หัวเรื่องย่อยลงมา (`<h2>`) แสดงหัวข้อประวัติ เช่น ชื่อเล่น, ว/ด/ป เกิด, อายุ, รหัสประจำตัวนักเรียน
3. ใช้แท็กย่อหน้า (`<p>`) และตัวขึ้นบรรทัดใหม่ (`<br/>`) บรรยายเหตุผลที่เลือกเรียนสาขานี้ และชื่อโรงเรียนเก่า
4. ใส่เส้นแบ่งหัวข้อ (`<hr/>`) คั่นในแต่ละส่วนของเนื้อหาให้ดูเรียบร้อย
5. บันทึกและทดสอบรันผลลัพธ์ผ่านเว็บเบราว์เซอร์ให้ถูกต้อง
