## 1. การปรับแต่งตัวอักษรหน้าเว็บ (CSS Typography)

การกำหนดสไตล์ตัวอักษรช่วยให้หน้าเว็บมีความสวยงามและเป็นระเบียบเรียบร้อยน่าอ่าน:

```css
/* ตัวอย่างการกำหนดคุณสมบัติข้อความใน stylesheet */
h1 {
    font-family: 'Prompt', 'Sarabun', sans-serif; /* กำหนดฟอนต์หลักและตระกูลสำรอง */
    font-size: 2rem;                              /* กำหนดขนาดแบบยืดหยุ่น */
    font-weight: 700;                             /* ความหนาอักษรระดับหนามาก */
    color: #333333;                               /* ปรับสีอักษร */
    text-align: center;                           /* จัดข้อความให้อยู่กึ่งกลางหน้าจอ */
    text-shadow: 2px 2px 4px rgba(0,0,0,0.1);     /* ใส่เงาตัวอักษรแบบจางๆ เพิ่มมิติ */
}
```

### คุณสมบัติการตกแต่งและจัดแนว (Other Text Properties):
*   `text-decoration: none;` ใช้สำหรับยกเลิกเส้นใต้ของแท็กลิงก์ `<a>` เพื่อไม่ให้ดูรบกวนสายตาผู้ใช้
*   `text-decoration: line-through;` ใช้สำหรับขีดเส้นฆ่าตรงกลางตัวอักษร (เช่น การลดราคาสินค้า ป้ายราคาเก่า)
*   **การตัดข้อความเป็นจุดไข่ปลา (...) เมื่ออักษรยาวล้นกล่อง:**
    ```css
    .ellipsis-text {
        white-space: nowrap;
        overflow: hidden;
        text-overflow: ellipsis;
    }
    ```

## 2. โครงสร้างแบบจำลองกล่อง CSS Box Model

องค์ประกอบของ HTML ทุกๆ ชิ้นส่วนจะถูกจำลองเป็น "กล่องสี่เหลี่ยม" ที่มีสัดส่วนความหนารวมทั้งหมด 4 ชั้น:

```
 ┌───────────────────────────────────────────┐
 │                Margin                     │  ← ระยะเว้นว่างห่างภายนอกกล่อง (ชนกับกล่องอื่น)
 │   ┌───────────────────────────────────┐   │
 │   │            Border                 │   │  ← เส้นขอบกล่อง (กำหนดความหนา สไตล์ สีได้)
 │   │   ┌───────────────────────────┐   │   │
 │   │   │        Padding            │   │   │  ← ฟองน้ำห่างภายใน (ระหว่างเนื้อหากับขอบกล่อง)
 │   │   │   ┌───────────────────┐   │   │   │
 │   │   │   │    Content        │   │   │   │  ← ตัวข้อมูลอักษรหรือภาพหลัก (width x height)
 │   │   │   │                   │   │   │   │
 │   │   │   └───────────────────┘   │   │   │
 │   │   └───────────────────────────┘   │   │
 │   └───────────────────────────────────┘   │
 └───────────────────────────────────────────┘
```

### รายละเอียดและคำสั่งจัดสไตล์:
1.  **Content (เนื้อหา):** กำหนดขนาดจริงผ่าน `width` (ความกว้าง) และ `height` (ความสูง)
2.  **Padding (ระยะห่างภายใน):** ควบคุมระยะห่างด้านในกล่อง
    *   `padding: 20px;` (เว้นรอบด้านเท่ากัน 20px)
    *   `padding: 10px 20px;` (เว้น บน-ล่าง 10px, ซ้าย-ขวา 20px)
    *   `padding-top`, `padding-bottom`, `padding-left`, `padding-right` (แยกเฉพาะด้าน)
3.  **Border (เส้นขอบ):**
    *   `border: 2px solid #ff0000;` (หนา 2px, เส้นลายทึบ solid, สีแดง)
    *   `border-radius: 8px;` (ปรับความโค้งมนของมุมขอบกล่อง)
    *   `border-radius: 50%;` (ดัดแปลงรูปทรงสี่เหลี่ยมจัตุรัสให้เป็นรูปวงกลมสมบูรณ์)
4.  **Margin (ระยะห่างภายนอก):** ควบคุมระยะที่เว้นไว้ให้ห่างจากวัตถุอื่น
    *   `margin: 20px;` (เว้นระยะภายนอก 20px รอบทิศ)
    *   `margin: 0 auto;` (บนล่างไม่เว้น, ซ้ายขวาปรับออโต้เพื่อ **จัดกล่องให้อยู่กึ่งกลางหน้าจอ**)

## 📝 ใบงานปฏิบัติท้ายบทเรียน (Assignment 6)

**โจทย์:** ให้นังเรียนจัดทำ "การ์ดแสดงราคาสินค้าขายรองเท้าพรีเมียม" ในไฟล์ชื่อ `product-card.html` โดยแสดงผลในรูปแบบกล่องการ์ดที่ตกแต่งให้สวยงามน่าดึงดูดด้วยกฎของ CSS Typography และ Box Model ดังนี้:
1.  **โครงสร้าง HTML:**
    *   สร้างกล่องครอบหลักด้วยแท็ก `<div>` กำหนดคลาสชื่อ `card`
    *   ด้านในใส่ชื่อยี่ห้อและรุ่นรองเท้า เช่น "PREMIUM SNEAKERS รุ่น X1"
    *   ใส่คำอธิบายรายละเอียดสั้นๆ ความยาวประมาณ 2-3 บรรทัด
    *   ใส่ราคาปกติ (ขีดฆ่าแสดงว่าลดราคา) และใส่ราคาสุดพิเศษตัวสีแดงสดหนาเข้ม
    *   สร้างปุ่มกดสั่งซื้อเป็นลิงก์หรือปุ่มระบุคำว่า "สั่งซื้อเลย"
2.  **การตกแต่ง CSS (แบบ Internal CSS หรือ External CSS):**
    *   *กล่องการ์ดหลัก (`.card`):* กำหนดความกว้างกล่อง (`width`) ขนาด 300px, ใส่สีพื้นหลังอ่อนๆ, กำหนดระยะ padding ด้านในกล่อง 20px เพื่อไม่ให้อักษรชนขอบ, ใส่ขอบรอบการ์ดหนา 1px สีเทาอ่อน และปรับมุมมน `border-radius: 12px;` พร้อมจัดกล่องให้อยู่กึ่งกลางหน้าจอด้วย `margin: 50px auto;`
    *   *หัวข้อรองเท้า:* เปลี่ยนฟอนต์อักษรเป็นตัวใหญ่เด่นชัด จัดให้อยู่กึ่งกลาง
    *   *รายละเอียดสินค้า:* เขียนคำสั่งตัดคำจุดไข่ปลา (`overflow: hidden; text-overflow: ellipsis; white-space: nowrap;`) ป้องกันข้อความยืดเพี้ยน
    *   *ปุ่มสั่งซื้อ:* ปรับพื้นหลังเป็นสีดำ ตัวอักษรสีขาว ลบเส้นใต้ลิงก์ออก (`text-decoration: none;`), ใส่ padding ข้างในปุ่มเพื่อให้ปุ่มดูหนาใหญ่สวยงาม และปรับมุมมนปุ่มเล็กน้อย
