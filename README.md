## 🛠️ เครื่องมือหลักที่ใช้ในรายวิชา
1. **VS Code (Visual Studio Code)**: โปรแกรมแก้ไขข้อความระดับมืออาชีพสำหรับใช้เขียน HTML, CSS และ JavaScript
2. **Google Chrome / Microsoft Edge**: เว็บเบราว์เซอร์หลักในการรันและดีบักโค้ดเพื่อแสดงผลหน้าเว็บ
3. **Live Server (Extension)**: ปลั๊กอินใน VS Code เพื่อใช้รันเซิร์ฟเวอร์จำลอง ช่วยรีเฟรชหน้าเบราว์เซอร์อัตโนมัติเมื่อกดเซฟไฟล์โค้ด
4. **Git / GitHub**: ใช้สำหรับส่งงาน ทำการบันทึกประวัติการแก้ไข และนำหน้าเว็บขึ้นสู่สาธารณะ
