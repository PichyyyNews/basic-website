ในสัปดาห์นี้เราจะเริ่มศึกษาแอตทริบิวต์ (Attributes) เพื่อใส่ข้อมูลรูปภาพและการสร้างลิงก์เชื่อมโยงจุดต่างๆ เรียนรู้การสร้างตารางข้อมูลเพื่อจัดระเบียบสารสนเทศ และทำความเข้าใจไวยากรณ์เบื้องต้นของ CSS เพื่อเริ่มทาสีสันให้หน้าเว็บเพจ

## 1. การใช้งานแท็กรูปภาพ `<img>` และนามสกุลไฟล์
แท็ก `<img>` ใช้สำหรับดึงภาพมาแสดงผล เป็นแท็กเดี่ยวที่ไม่จำเป็นต้องมีคู่แท็กปิด

```html
<img src="images/profile.webp" alt="รูปโปรไฟล์นักเรียนปวช 1 สาขาไอที" width="200" height="200" loading="lazy" title="คลิกเพื่อขยายดูรูปภาพ">
```

### รายละเอียด Attributes สำคัญ:
1.  `src` (Source): ระบุตำแหน่งไฟล์รูปภาพในเครื่องหรือลิงก์อินเทอร์เน็ต
2.  `alt` (Alternative Text): ข้อความบรรยายภาพ จะแสดงผลเมื่ออินเทอร์เน็ตโหลดภาพไม่สำเร็จ และเป็นส่วนสำคัญในการดึงระบบค้นหาเข้ามาอ่านเนื้อหา (SEO)
3.  `width` / `height`: ควบคุมความกว้างและสูง
4.  `loading="lazy"`: สั่งเบราว์เซอร์เลื่อนจอมาถึงแล้วค่อยดาวน์โหลดภาพ ช่วยให้หน้าแรกของเว็บแสดงผลได้เร็วขึ้น

## 3. การสร้างตารางข้อมูลด้วย HTML `<table>`
การสร้างตารางแบบมีแถวหัวเรื่อง แถวเนื้อหา และแถวท้ายตาราง เพื่อแสดงข้อมูลอย่างเป็นระบบ:

```html
<table>
    <caption>ตารางสรุปเกรดเฉลี่ยสะสม</caption>
    <thead>
        <tr>
            <th>รหัสวิชา</th>
            <th>ชื่อวิชา</th>
            <th>หน่วยกิต</th>
            <th>เกรด</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>21909-1006</td>
            <td>พื้นฐานการสร้างเว็บไซต์</td>
            <td>2</td>
            <td>4.0</td>
        </tr>
        <tr>
            <td>31909-2007</td>
            <td>การประมวลผลภาพดิจิทัล</td>
            <td>3</td>
            <td>3.5</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <th colspan="2">รวมหน่วยกิต / เกรดเฉลี่ย</th>
            <td>5</td>
            <td>3.7</td>
        </tr>
    </tfoot>
</table>
```

## 4. วิธีการต่อใช้งาน CSS 3 รูปแบบ

### วิธีที่ 1: Inline CSS (เขียนตรงในแท็กเดี่ยว)
เหมาะสมสำหรับทดลองตกแต่งเฉพาะจุดสั้นๆ อย่างเร่งด่วน:
```html
<h1 style="color: blue; text-align: center;">หัวข้อสีน้ำเงินจัดตรงกลาง</h1>
```

### วิธีที่ 2: Internal CSS (เขียนในแท็ก `<style>` ด้านบน)
สะดวกสำหรับเว็บหน้าเดียวที่ต้องการรูปแบบตกแต่งพิเศษเฉพาะเจาะจงไม่ซ้ำกับหน้าอื่น:
```html
<head>
    <style>
        body {
            background-color: #f9f9f9;
            font-family: 'Sarabun', sans-serif;
        }
        table, th, td {
            border: 1px solid black;
        }
    </style>
</head>
```

### วิธีที่ 3: External CSS (การแยกเขียนไฟล์ `.css` ต่างหาก - วิธีแนะนำ)
เขียนแต่งทั้งหมดไว้ในไฟล์ `style.css` แล้วดึงลิงก์เข้ามาใช้งานที่หน้า HTML:
```html
<!-- ดึงลิงก์แต่งสไตล์ในแท็ก <head> -->
<link rel="stylesheet" href="style.css">
```
