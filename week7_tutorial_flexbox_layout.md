# คู่มือปฏิบัติการ Week 7: การจัดหน้าเว็บเพจยืดหยุ่นด้วย CSS Flexbox

ในสัปดาห์นี้เราจะเรียนรู้ระบบจัดหน้าเว็บเพจที่สำคัญที่สุดในปัจจุบันคือ **CSS Flexbox** ซึ่งเป็นเทคนิคการจัดการเลย์เอาต์ (Layout) ของวัตถุต่าง ๆ บนหน้าเว็บเพจให้มีความยืดหยุ่น ปรับเปลี่ยนตำแหน่ง และแสดงผลให้สอดคล้องตามขนาดหน้าจอของอุปกรณ์ต่าง ๆ ได้อย่างดีเยี่ยม

---

## 1. แนวคิด Flex Container และ Flex Items

เมื่อเราต้องการจัดวางสิ่งของ เราจะแบ่งบทบาทออกเป็น 2 ชั้น:
1.  **Flex Container (กล่องแม่):** กล่อง HTML ที่เราเปิดคำสั่ง `display: flex;` คลุมไว้ด้านนอก
2.  **Flex Items (กล่องลูก):** แท็ก HTML ทุกตัวที่เป็นลูกโดยตรงของกล่องแม่ จะเปลี่ยนพฤติกรรมมาเรียงตัวตามที่เราต้องการทันที

```html
<!-- โครงสร้างใน HTML -->
<div class="flex-container">
    <div class="flex-item">ชิ้นส่วนที่ 1</div>
    <div class="flex-item">ชิ้นส่วนที่ 2</div>
</div>
```

---

## 2. คุณสมบัติควบคุมเลย์เอาต์บน Flex Container (กล่องแม่)

### 2.1 display: flex;
ระบุในคลาสของกล่องแม่เพื่อสั่งเริ่มใช้งานระบบ Flexbox

### 2.2 flex-direction (ทิศทางแนวแกนเรียงสิ่งของ)
*   `flex-direction: row;` (ค่าดีฟอลต์) เรียงกล่องลูกเป็นแนวนอนจากซ้ายไปขวา
*   `flex-direction: column;` เรียงกล่องลูกเป็นแนวตั้งจากบนลงล่าง

### 2.3 gap (ระยะห่างช่องไฟ)
ใช้ระบุระยะช่องว่างห่างคงที่ระหว่างกล่องลูกแต่ละใบ เช่น `gap: 20px;` สะดวกกว่าการกำหนด margin ในกล่องลูกทีละใบ

### 2.4 justify-content (จัดแนวแกนหลัก - แนวนอนเมื่อเป็น row)
*   `justify-content: flex-start;` เรียงชิดขอบจุดเริ่มต้นด้านซ้าย
*   `justify-content: center;` เรียงกองรวมกันอยู่ตรงกลาง
*   `justify-content: flex-end;` เรียงชิดขอบท้ายด้านขวา
*   `justify-content: space-between;` ขยายระยะห่างสิ่งของให้เท่าๆ กัน โดยตัวแรกและตัวสุดท้ายติดริมขอบข้าง (นิยมใช้สร้าง Navigation Bar)
*   `justify-content: space-around;` เฉลี่ยช่องว่างโดยรอบทุกกล่องให้เท่ากัน

### 2.5 align-items (จัดแนวแกนรอง - แนวตั้งเมื่อเป็น row)
*   `align-items: flex-start;` ชิดด้านบนสุด
*   `align-items: center;` จัดข้อความหรือสิ่งของให้อยู่กึ่งกลางความสูงของกล่องแม่
*   `align-items: stretch;` (ค่าดีฟอลต์) ยืดขนาดความสูงของกล่องลูกให้สูงสุดเท่ากับขนาดความสูงของกล่องแม่

### 2.6 flex-wrap (ควบคุมการขึ้นบรรทัดใหม่เมื่อล้นจอ)
*   `flex-wrap: nowrap;` (ค่าดีฟอลต์) ฝืนอัดให้อยู่ในแถวเดียว ไม่มีการแยกแถว
*   `flex-wrap: wrap;` ตัดส่วนที่เกินออกไปไหลขึ้นบรรทัดใหม่ด้านล่างทันที (จำเป็นมากสำหรับรองรับหน้าจอมือถือ)

## 3. ตัวอย่างการประยุกต์ใช้งาน Flexbox

### 3.1 การสร้างแถบเมนูนำทางด้านบน (Header Navigation Bar):
```html
<header style="display: flex; justify-content: space-between; align-items: center; background-color: #333; padding: 15px 30px; color: white;">
    <div class="logo">IT SHOP</div>
    <nav style="display: flex; gap: 20px;">
        <a href="#" style="color: white; text-decoration: none;">หน้าแรก</a>
        <a href="#" style="color: white; text-decoration: none;">สินค้า</a>
        <a href="#" style="color: white; text-decoration: none;">ติดต่อเรา</a>
    </nav>
</header>
```

## 📝 ใบงานปฏิบัติท้ายบทเรียน (Assignment 7)

**โจทย์:** ให้นักเรียนสร้างระบบแผงหน้าจอรายการแสดงสินค้าชื่อ `shop-layout.html` โดยมีคุณสมบัติต่างๆ ดังนี้:
1.  **แถบส่วนหัว (Header):**
    *   สร้างกล่องแถบนำทางส่วนหัว ใช้ CSS Flexbox จัดสไตล์ให้โลโก้อยู่ทางฝั่งซ้ายสุด และเมนูลิงก์นำทาง (หน้าแรก, สินค้า, เกี่ยวกับเรา) อยู่ทางฝั่งขวาสุด (`justify-content: space-between;` และ `align-items: center;`)
    *   ตกแต่งสีสันพื้นหลังและขนาดตามความเหมาะสม
2.  **แผงรายการสินค้า (Product Grid):**
    *   สร้างกล่องครอบขนาดใหญ่สำหรับใส่การ์ดสินค้าอย่างน้อย 3 ใบ
    *   กำหนดสไตล์กล่องครอบหลักให้เป็น Flexbox เรียงตัวแนวนอน (`flex-direction: row;`) มีระยะห่างระหว่างสินค้า `gap: 20px;` และสั่งให้กล่องสินค้าไหลลงมาจัดเรียงใหม่เมื่อจอย่อแคบลงด้วย `flex-wrap: wrap;`
3.  **การ์ดสินค้าแต่ละชิ้น:**
    *   นำทักษะสัปดาห์ที่ 6 มาช่วยตกแต่งขนาด ปรับแต่งสีสัน ใส่กรอบ ขอบมน และเงาของการ์ดให้ดูดีมีระดับแบบเว็บจริง
4.  ตรวจสอบและยืดหดทดสอบขนาดหน้าต่างเบราว์เซอร์ เพื่อยืนยันว่าเมื่อขนาดย่อเล็กลงการ์ดสินค้าจะไหลลงมาสแต็กเรียงต่อในแนวตั้งแบบอัตโนมัติ (Responsive Layout)
