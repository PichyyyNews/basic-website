## 1. การใช้งานเครื่องมือพัฒนา Visual Studio Code (VS Code)

### 1.1 ปลั๊กอิน Extension แนะนำ:
1.  **HTML5 Boilerplate:** ตัวช่วยสร้างโครงร่างมาตรฐาน HTML5 ครบถ้วนอัตโนมัติ
2.  **Live Server:** ตัวสร้าง Local Web Server บนเครื่องเพื่อรันเบราว์เซอร์และรีเฟรชหน้าผลลัพธ์ทันทีเมื่อกดบันทึกโค้ด (`Ctrl + S`)

### 1.2 คีย์ลัดแป้นพิมพ์พื้นฐานที่โปรแกรมเมอร์ใช้บ่อย:
*   `Ctrl + P`: ค้นหาไฟล์ในโปรเจกต์อย่างรวดเร็ว
*   `Ctrl + B`: ปิด-เปิด เมนูแถบจัดการไฟล์ด้านซ้ายมือ
*   `Ctrl + /`: เปิดหรือปิดข้อความคอมเมนต์ (Comment) โค้ดบรรทัดนั้น
*   `Shift + Alt + F`: จัดรูปแบบหน้าตาการย่อหน้าเคาะเคอร์เซอร์ให้เป็นระเบียบสวยงามตามมาตรฐาน (Format Document)

## 2. การใช้งานแท็กมัลติมีเดียเล่นวิดีโอและเสียง

### 2.1 แท็กวิดีโอ `<video>` (Video Element)
การใส่ไฟล์วิดีโอมาตรฐานลงบนหน้าเว็บเพจ:

```html
<video width="640" height="360" controls poster="images/cover.jpg" playsinline preload="metadata">
    <source src="media/promo.mp4" type="video/mp4">
    <source src="media/promo.webm" type="video/webm">
    เบราว์เซอร์ของคุณไม่รองรับการเล่นไฟล์วิดีโอนี้
</video>
```

#### รายละเอียด attributes:
*   `controls`: บังคับแสดงปุ่มกด Play/Pause และแถบเลื่อนเวลา
*   `poster`: ระบุที่อยู่ไฟล์ภาพที่จะเอามาแสดงเป็นหน้าปกก่อนกดเล่น
*   `playsinline`: ป้องกันวิดีโอเด้งเต็มจอภาพอัตโนมัติเมื่อดูบนมือถือ (เช่น บนระบบ iOS)
*   `preload="metadata"`: โหลดข้อมูลขนาดและความยาวของไฟล์มารอไว้ก่อน เพื่อลดการกินอินเทอร์เน็ตของผู้ใช้

### 2.2 แท็กเสียง `<audio>` (Audio Element)
```html
<audio controls loop muted>
    <source src="media/ambient.mp3" type="audio/mpeg">
    เบราว์เซอร์ของคุณไม่รองรับไฟล์เสียงนี้
</audio>
```

## 3. ความแตกต่างของกล่อง Div และอักษร Span (Block vs Inline)

1.  **แท็ก `<div>` (Block Element):**
    *   ทำหน้าที่สร้าง "กล่องเลย์เอาต์ขนาดใหญ่" 
    *   คุณสมบัติทางพฤติกรรม: **ขึ้นบรรทัดใหม่ทันที** และ **ยึดความกว้างเต็มจอ (100%)**
2.  **แท็ก `<span>` (Inline Element):**
    *   ทำหน้าที่สร้าง "กลุ่มอักษรเฉพาะจุด" ภายในย่อหน้าหรือภายในบรรทัดเดียวกัน
    *   คุณสมบัติทางพฤติกรรม: **ไม่ขึ้นบรรทัดใหม่** และ **ยึดพื้นที่จำกัดเฉพาะเท่าขนาดตัวอักษรจริง**

```html
<!-- ตัวอย่างความแตกต่าง -->
<div>กล่องใบที่ 1 (Block)</div>
<div>กล่องใบที่ 2 (Block) จะแยกบรรทัดทันที</div>

<p>นี่คือข้อความในบรรดาย่อหน้าและต้องการเน้นคำว่า <span style="color: red;">ภาษาไทย</span> ให้เป็นสีแดงโดยไม่กระโดดขึ้นบรรทัดใหม่</p>
```

## 4. ความรู้พื้นฐานเกี่ยวกับ CSS Selectors (ตัวเลือกสไตล์)

เพื่อควบคุมว่ารหัสสไตล์ CSS จะไปจัดแบบแผนหน้าตาให้ตรงกับแท็กไหนใน HTML:

1.  **Element Selector (เลือกตามชื่อแท็ก):** ควบคุมทุกแท็กชนิดนั้นๆ ทั้งหน้าเพจ
    ```css
    p {
        color: blue; /* ทุกแท็ก p จะกลายเป็นสีน้ำเงิน */
    }
    ```
2.  **Class Selector (เลือกตามชื่อคลาส):** สไตล์กลุ่มที่สามารถนำไปใช้ซ้ำได้หลายแท็ก โดยมีจุด `.` นำหน้า
    ```css
    .text-green {
        color: green; /* แท็กใดๆ ที่ใช้ class="text-green" จะเป็นสีเขียว */
    }
    ```
3.  **ID Selector (เลือกเจาะจงเฉพาะตัว):** ใช้สัญลักษณ์ `#` นำหน้า มีผลเฉพาะกล่องเดียวที่มี `id` ตรงกันในหน้าเพจ ห้ามซ้ำ
    ```css
    #main-logo {
        width: 150px;
    }
    ```
