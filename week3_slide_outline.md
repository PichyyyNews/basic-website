## Slide 2: วัตถุประสงค์การเรียนรู้
*   ใช้งานแท็กรูปภาพ `<img>` พร้อมแอตทริบิวต์หลักและจัดระบบ SEO เบื้องต้นได้

*   จัดระเบียบตารางข้อมูลพื้นฐานด้วยกลุ่มแท็ก `<table>` ได้ถูกต้อง
*   เข้าใจแนวคิดการใช้งาน CSS และบอกโครงสร้างการเชื่อมต่อสไตล์ทั้ง 3 แบบได้

## Slide 3: แอตทริบิวต์ (Attributes) ใน HTML คืออะไร?
*   ส่วนขยายหรือค่ากำหนดเพิ่มเติมที่เขียนไว้ด้านใน "แท็กเปิด" เสมอ
*   รูปแบบการเขียน: `name="value"`
*   *ตัวอย่าง:* `<tagname name="value">เนื้อหา</tagname>`
*   *เช่น:* `<div id="header" class="container">...</div>`

## Slide 4: แท็กแสดงผลรูปภาพ `<img>`
*   เป็นแท็กประเภท **Empty Element** (แท็กเดี่ยว ไม่มีแท็กปิด `</img>`)
*   **Required Attributes (แอตทริบิวต์บังคับ):**
    *   `src` (Source): ระบุที่อยู่ไฟล์ภาพ (Relative หรือ Absolute path)
    *   `alt` (Alternative Text): คำอธิบายรูปภาพสำหรับระบบค้นหา (SEO) และเครื่องอ่านจอภาพ
*   **Optional Attributes (ตัวเลือกเสริม):**
    *   `width` / `height`: ปรับขนาดภาพ (หน่วยเป็นพิกเซลโดยปริยาย)
    *   `loading="lazy"`: เลื่อนจอดึงข้อมูลเฉพาะเมื่อจำเป็น ช่วยให้เว็บทำงานรวดเร็วขึ้น
    *   `title`: แสดงข้อความ Tooltip เมื่อเอาเมาส์ไปชี้ที่รูปภาพ

## Slide 5: นามสกุลไฟล์รูปภาพที่ควรเลือกใช้
*   **WebP:** นามสกุลสมัยใหม่ แนะนำเป็นหลักเพราะไฟล์มีขนาดเล็กมากแต่ยังคงคมชัดสูง
*   **JPEG:** เหมาะสำหรับภาพถ่ายทั่วไปที่มีความสลับซับซ้อนของสีสูง
*   **PNG:** เหมาะสำหรับรูปภาพที่ต้องการเจาะทะลุโปร่งใส (Transparent Background)
*   **SVG:** เหมาะสำหรับภาพลายเส้น โลโก้แบรนด์ ขยายเท่าไหร่ก็ไม่แตกเบลอ

## Slide 6: แท็กสร้างลิงก์เชื่อมโยง `<a>` (Anchor Tag)
*   ใช้สร้างช่องทางไฮเปอร์ลิงก์ไปยังที่ต่างๆ 
*   **แอตทริบิวต์ที่สำคัญ:**
    *   `href` (Hypertext Reference): ระบุปลายทาง
        *   เชื่อมโยงภายนอก: `href="https://google.com"`
        *   เชื่อมโยงภายใน: `href="about.html"`
        *   ลิงก์อีเมล: `href="mailto:support@company.com"`
        *   ลิงก์เบอร์โทร: `href="tel:+66812345678"`
    *   `target="_blank"`: เปิดลิงก์เชื่อมโยงบนแท็บหน้าต่างเบราว์เซอร์อันใหม่

## Slide 7: แท็กสร้างตารางข้อมูล `<table>`
*   แสดงผลเปรียบเสมือนสมุดบันทึกหรือไฟล์ Excel
*   **ส่วนประกอบโครงสร้าง:**
    *   `<table>`: กรอบหลักของตาราง
    *   `<caption>`: ชื่อหรือคำอธิบายหน้าตาราง
    *   `<tr>` (Row): แถวตาราง
    *   `<th>` (Header): เซลล์หัวข้อคอลัมน์ (แสดงผลตัวหนาอยู่ตรงกลาง)
    *   `<td>` (Data Cell): เซลล์เนื้อหาช่องข้อมูลทั่วไป

## Slide 8: CSS คืออะไร?
*   **CSS (Cascading Style Sheets):** ทำหน้าที่เป็น "ดีไซเนอร์" คอยตกแต่งโครงสร้างบ้านให้มีสีสัน ทาสี ปรับหน้าตาให้สวยงาม
*   หากไม่มี CSS หน้าเว็บจะมีเพียงข้อความแถวแนวตั้งสีดำบนพื้นหลังสีขาวธรรมดา
*   **โครงสร้างไวยากรณ์ (Syntax):**
    `Selector { Property: Value; }`
    *ตัวอย่าง:* `h1 { color: blue; font-size: 16px; }`

## Slide 9: วิธีการเขียน CSS ทั้ง 3 วิธี
1.  **Inline CSS:** เขียนตรงในแท็ก HTML
    `<h1 style="color: blue; text-align: center;">...</h1>`
2.  **Internal CSS:** เขียนในแท็ก `<style>` ด้านใน `<head>`
    `<style> body { background-color: linen; } </style>`
3.  **External CSS (วิธีที่ดีที่สุด):** แยกไปเขียนที่ไฟล์ `.css` เช่น `style.css` แล้วเชื่อมเข้ามาผ่านแท็ก `<link>` ในส่วนหัวเรื่อง
