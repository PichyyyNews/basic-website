## 1. ข้อแตกต่างระหว่างการเขียนแบบ Semantic HTML กับการเขียนแบบแนะนำผิดวิธี (Abuse for Styling)

*   **Abuse for Styling:** การนำแท็ก `<h1>` หรือ `<h2>` ไปครอบข้อความธรรมดาทั่วไปเพียงเพราะต้องการให้ข้อความนั้นแสดงผลเป็น "ตัวหนา" และ "ตัวใหญ่" แม้ว่าจุดนั้นไม่ใช่หัวข้อสำคัญจริง ๆ 
*   **Semantic HTML:** การใช้ `<h1>` เพียงครั้งเดียวเป็นชื่อเรื่องหลักของหน้าเว็บ จากนั้นใช้ `<h2>` และ `<h3>` ไล่ระดับลงมาตามลำดับความสำคัญ ห้ามข้ามสเต็ป (เช่น ข้ามจาก `<h1>` ไปเป็น `<h4>` เลย) หากต้องการให้ข้อความธรรมดาตัวใหญ่ ให้เขียนแต่งด้วย CSS สไตล์ในสัปดาห์ต่อๆ ไป

## 2. ตารางและคู่มือแท็กจัดสไตล์อักษรเฉพาะจุด (Text Formatting)

| แท็ก HTML | ชื่อเต็ม | ความหมายและผลลัพธ์บนเบราว์เซอร์ |
|:---:|---|---|
| `<b>` | Bold | ตัวอักษรหนา (แสดงผลเน้นเข้มทั่วไป) |
| `<strong>` | Strong Importance | ตัวอักษรหนา (สื่อว่าจุดนี้มีความสำคัญสูง) |
| `<i>` | Italic | ตัวอักษรเอียง (เน้นภาพรวมทั่วไป) |
| `<em>` | Emphasized | ตัวอักษรเอียง (เน้นย้ำความหมายของประโยค) |
| `<mark>` | Marked | ไฮไลต์พื้นหลังคำเป็นสีเหลืองสว่าง |
| `<small>` | Small | ตัวอักษรย่อขนาดให้เล็กกว่าปกติหนึ่งระดับ |
| `<del>` | Deleted | ตัวอักษรแบบมีเส้นขีดฆ่าตรงกลางตัว |
| `<ins>` | Inserted | ตัวอักษรแบบขีดเส้นใต้คำ |
| `<sub>` | Subscript | ตัวอักษรห้อยลงครึ่งบรรทัดด้านล่าง |
| `<sup>` | Superscript | ตัวอักษรยกลอยขึ้นครึ่งบรรทัดด้านบน |

### ตัวอย่างการใช้ตัวห้อยและตัวยก (Subscript & Superscript):
*   **ก๊าซคาร์บอนไดออกไซด์:** `CO<sub>2</sub>` -> แสดงผลเป็น CO₂
*   **น้ำ:** `H<sub>2</sub>O` -> แสดงผลเป็น H₂O
*   **ประจุของซัลเฟตไอออน:** `SO<sub>4</sub><sup>2-</sup>` -> แสดงผลเป็น SO₄²⁻
*   **คณิตศาสตร์สมการกำลังสอง:** `x<sup>2</sup> + y<sup>2</sup> = z<sup>2</sup>` -> แสดงผลเป็น x² + y² = z²

## 3. การสร้างรายการข้อความ (HTML Lists)

### 3.1 Unordered List `<ul>` (รายการไม่มีตัวเลขกำกับ)
ใช้สัญลักษณ์สากล เช่น จุดกลมดำ (Disc) นำหน้าข้อความ:
```html
<ul>
    <li>วิชาภาษาไทย</li>
    <li>วิชาคณิตศาสตร์</li>
    <li>วิชาพื้นฐานการสร้างเว็บไซต์</li>
</ul>
```

### 3.2 Ordered List `<ol>` (รายการมีตัวเลขหรือลำดับอักษรเรียงกัน)
จัดลำดับก่อนหลังอย่างชัดเจน:
```html
<ol>
    <li>เปิดคอมพิวเตอร์</li>
    <li>เปิดโปรแกรม Notepad</li>
    <li>เริ่มต้นเขียนโค้ด</li>
</ol>
```

### 3.3 Description List `<dl>` (รายการนิยามศัพท์)
ใช้จัดระเบียบเนื้อหาประเภทพจนานุกรมศัพท์หรือคำถาม-คำตอบ:
```html
<dl>
    <dt>Domain Name</dt>
    <dd>- ชื่อที่ใช้ระบุตัวแทนระบบไอพีแอดเดรสของเว็บไซต์</dd>
    <dt>Web Hosting</dt>
    <dd>- เครื่องคอมพิวเตอร์เซิร์ฟเวอร์ที่เก็บข้อมูลหน้าเว็บ</dd>
</dl>
```
