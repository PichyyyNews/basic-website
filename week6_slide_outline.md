## Slide 1: หัวข้อหลัก
*   **หัวข้อ:** การตกแต่งตัวอักษรหน้าเว็บ และทฤษฎีแบบจำลองกล่อง (Box Model)
*   **วิชา:** 21909-1006 พื้นฐานการสร้างเว็บไซต์ (ปวช. 1)

## Slide 2: วัตถุประสงค์การเรียนรู้
*   ใช้งานคุณสมบัติ CSS เพื่อกำหนดสเปกตัวอักษรและแบบฟอนต์ได้ (Typography styles)
*   เขียนคำสั่งตัดข้อความอักษรล้นกล่องด้วยจุดไข่ปลา (...) ได้
*   อธิบายองค์ประกอบทั้ง 4 ชั้นของแบบจำลองกล่อง (CSS Box Model) ได้อย่างเข้าใจ
*   ประยุกต์ใช้ Box Model ในการจัดระยะห่าง ขนาดกล่อง และขอบมนของการ์ดขายสินค้าได้

## Slide 3: การจัดการฟอนต์และตัวอักษร (Typography)
*   **font-family:** เปลี่ยนสไตล์รูปแบบฟอนต์ นิยมใส่รายชื่อฟอนต์หลักตามด้วยฟอนต์ตระกูลสำรอง เช่น `'Sarabun', sans-serif`
*   **font-size:** ขนาดอักษร (หน่วย px ขนาดคงที่ หรือ rem ยืดหยุ่นตามฐานรากเว็บ)
*   **font-weight:** ความหนาอักษร (คำศัพท์: bold, normal หรือตัวเลข: 100-900)
*   **font-style:** ปรับตัวเอียง (`italic`) หรือปกติ (`normal`)
*   **color:** สีตัวอักษร

## Slide 4: การจัดสไตล์ตกแต่งข้อความเพิ่มเติม
*   **text-decoration:** ขีดตกแต่งอักษร
    *   `none`: ลบเส้นใต้เดิม (ใช้บ่อยเพื่อจัดหน้าลิงก์ `<a>` ไม่ให้รกสายตา)
    *   `underline`: ขีดเส้นใต้
    *   `line-through`: ขีดเส้นฆ่าตรงกลางอักษร (ทำราคาขายสินค้าลดพิเศษ)
*   **text-transform:** เปลี่ยนพิมพ์เล็ก/ใหญ่ของภาษาอังกฤษ (`uppercase`, `lowercase`, `capitalize`)
*   **text-shadow:** ทำเงาให้อักษรเพิ่มมิติความลึก (แกน X, แกน Y, ความฟุ้งเบลอ, สีเงา)

## Slide 5: ปัญหาอักษรล้นกล่อง (Text Overflow)
*   เมื่อข้อความยาวเกินช่องกล่องที่เรากำหนดไว้ จะทำให้เลย์เอาต์หน้าจอฉีกเพี้ยน
*   **แนวทางแก้ไข (ทำจุดไข่ปลา ...):**
    ```css
    .ellipsis {
        white-space: nowrap;      /* สั่งห้ามหักบรรทัดใหม่ */
        overflow: hidden;         /* ซ่อนคำส่วนที่เกินขอบ */
        text-overflow: ellipsis;  /* แสดง ... ท้ายข้อความที่โดนตัด */
    }
    ```

## Slide 6: กฎของแบบจำลองกล่อง CSS Box Model
*   ในระบบเบราว์เซอร์ ทุกชิ้นส่วนองค์ประกอบ HTML จะมีกรอบสี่เหลี่ยมห่อหุ้ม 4 ชั้น
*   ประกอบด้วย:
    1.  **Content:** ตัวเนื้อหาหลัก (ควบคุมขนาดด้วย `width` และ `height`)
    2.  **Padding:** ระยะห่างภายใน (ขอบเขตระหว่างเนื้อหาถึงเส้นขอบ)
    3.  **Border:** เส้นขอบกล่อง (ความหนา ลายเส้น สี)
    4.  **Margin:** ระยะห่างภายนอก (ระยะที่กล่องนี้เว้นเผื่อสำหรับกล่องข้างๆ)

## Slide 7: เจาะลึก Padding vs Margin
*   **Padding (ระยะห่างด้านใน):** เปรียบเสมือนโฟมฟองน้ำซับแรงกระแทก ยิ่งขยาย ค่ากล่องจะมีพื้นที่ด้านในสว่างขึ้นและกว้างขึ้น
*   **Margin (ระยะห่างด้านนอก):** เปรียบเสมือนแนวป้องกันรักษาระยะห่าง ไม่ให้สิ่งอื่นมาเบียดทับ ไม่เพิ่มพื้นที่ด้านในกล่อง

## Slide 8: ขอบมนและการทำรูปทรงกลม
*   **border-radius:** ปรับให้กรอบกล่องโค้งมนสวยงาม
    *   ตัวอย่าง: `border-radius: 8px;` ปรับขอบมนเล็กน้อย
    *   ตัวอย่าง: `border-radius: 50%;` ปรับเป็นวงกลมสมบูรณ์แบบ (เมื่อกว้างสูงของกล่องเท่ากัน)
